# 🐨 Koalla.ai

> Registre. Organize. Encontre Tudo.

Koalla.ai é um assistente de IA que captura e organiza informações diretamente pelo WhatsApp. Envie um texto, uma nota de voz ou uma foto — o Koalla classifica, categoriza e armazena tudo automaticamente para você encontrar quando precisar.

## Como funciona

1. **Envie pelo WhatsApp** — texto, áudio ou foto, como preferir
2. **A IA processa e organiza** — em segundos, classifica e categoriza o conteúdo
3. **Encontre quando precisar** — busca por texto, filtros por categoria, data ou tag

## Funcionalidades

- **Transcrição de áudio** — notas de voz viram texto organizado automaticamente
- **Categorização automática** — a IA identifica o tipo de informação (vendas, agenda, tarefas, ideias, urgente...)
- **Busca instantânea** — encontre qualquer registro em segundos
- **Histórico completo** — linha do tempo com data, hora e contexto de cada registro
- **Relatórios e exportação** — gere resumos e exporte seus dados com um comando
- **Segurança** — dados criptografados e privados
